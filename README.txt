## Extentions requises : 
asgiref==3.8.1
Django==5.1.2
sqlparse==0.5.1
typing_extensions==4.12.2
tzdata==2024.2
Peuvent être installées avec pip install django.

Pour lancer le serveur, placez vous dans le repertoire site_reservation_velo,
exectuez la commande : "python manage.py runserver"

accedez dans nimporte quel navigateur à l'adresse: "http://127.0.0.1:8000/reservations/"
